For running quick_test.py script, first create a python enviroment and install the requirements. For this project, I used Python 3.6.8.

Then in the root directory of the bench repository: run `.\_pyenv_Autoformer\Scripts\python.exe Autoformer\quick_test.py`. This `.\_pyenv_Autoformer\Scripts\python.exe` is the location where I keep the python environment for Autoformer.
